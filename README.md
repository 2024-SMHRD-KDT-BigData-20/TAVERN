# 핵심프로젝트
양주소개 커뮤니티

## 팀이름
TARVERN : 선술집이라는 의미

## 팀구성
김수빈(팀장, 총괄 밑 데이터전처리), 박승현(프론트), 강규현(백엔드), 강승완(백엔드)

## 화면 설계
메인페이지, 정보페이지, 정보 상세페이지, 글쓰기페이지, 아이디비번찾기, 커뮤니티메인페이지, 커뮤니티자유, 커뮤니티리뷰, 커뮤니티상세, 문의, 문의글쓰기, 소식및이벤트
